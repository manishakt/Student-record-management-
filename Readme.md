## Project Description

The **Student Record Management System** is a console-based application developed in **C++**. It is designed to efficiently manage student information such as student ID, name, age, course, and marks. The project demonstrates the use of Object-Oriented Programming (OOP) concepts, including classes, objects, file handling, and menu-driven programming.

## Features

- Add a new student record
- View all student records
- Search for a student by ID
- Update student information
- Delete a student record
- Store records using file handling

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Console Application

## How to Run

1. Compile the source code using any C++ compiler (such as g++).
2. Run the generated executable file.
3. Use the menu options to manage student records.

## Output

The program provides a simple menu-driven interface for adding, viewing, searching, updating, and deleting student records.
